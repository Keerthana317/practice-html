Types of Styling

We have the 3 types of Stylings are there: 

 
                                      1. Inline Styling
                                      2. Internal Styling
                                      3. External Styling

1. Inline Styling

In inline styling by using "style attribute" in HTML element.

2. Internal Styling

In internal styling by using "style as the tag" in HTML element.

3. External Styling

In external styling by using "link element" to link an external css file.

Types of Selectors

CSS selectors are used to find the HTML elements to style. There are 6 types of selectors are:


                    1. Universal Selectors
                    2. Tag Selectors
                    3. Grouping Selectors
                    4. Class Selectors
                    5. ID Selectors
